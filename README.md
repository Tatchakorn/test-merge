# Testing Implementation of Merge

Given two sorted arrays `collection_1` and `collection_2`
```go
merge (int[] collection_1, int[] collection_2) : int []
```

## Install Dependencies
### with NPM
```bash
npm install
```
### with Yarn
```
yarn
```

## Run Test
### with NPM
```
npm run build
npm test
```
### with Yarn
```
yarn build
yarn test
```
